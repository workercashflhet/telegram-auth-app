# Telegram Auth App

Простое мини-приложение для Telegram с авторизацией через Telegram и выводом информации пользователя.

## Функционал
- Авторизация через Telegram Web App
- Получение данных пользователя из Telegram
- Отображение всей доступной информации о пользователе

cd telegram-auth-app
